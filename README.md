# Taste of Home - Luxury Restaurant Website

A stunning, premium front-end application built for a high-end luxury Indian restaurant, "Taste of Home". This application showcases the restaurant's legacy, exquisite menu, beautiful interior ambiance, and highly acclaimed culinary team.

## рџЊџ Key Features

- **Premium UI/UX:** A bespoke deep-dark aesthetic with elegant gold accents, reflecting the luxury of the dining experience.
- **Smooth Animations:** Powered by Framer Motion, the site features elegant scroll animations, parallax effects, and smooth page transitions.
- **Dynamic Routing:** Multi-page architecture including Home, Menu, About, Contact, Gallery, and Testimonials.
- **Responsive Design:** Fully responsive layout that looks stunning on mobile, tablet, and desktop devices.
- **Interactive Elements:** Glowing neon social media icons, animated image hover effects, and beautifully aligned components.

## рџљЂ Getting Started

### Prerequisites
Make sure you have Node.js and npm installed on your machine.

### Installation

1. **Clone the repository or navigate to the project directory:**
   ```bash
   cd restaurant
   ```

2. **Navigate to the frontend folder:**
   ```bash
   cd frontend
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to the local URL provided in your terminal (usually `http://localhost:5173`).

## рџЋЁ Design Philosophy

The design revolves around a `deep dark` theme (`#0a0a0a`) contrasted with a signature `primary gold` (`#D4AF37`). Every component is built with an emphasis on:
- High-quality imagery from Unsplash showcasing real luxury interiors and chefs.
- Elegant Serif typography (`Great Vibes` for signatures, elegant Serif for quotes).
- Minimalist spacing to let the content breathe.
